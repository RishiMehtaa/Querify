This repository is a prototype implementation of "querify" intended to validate ideas for query construction, transformation, and execution. It focuses on fast iteration to explore architecture and core behaviors rather than production readiness.

Key goals:
- Demonstrate core query-generation and transformation flows.
- Integrate with one or more data sources to exercise end-to-end behavior.
- Provide a lightweight API/CLI and example usage to evaluate design choices.
- Surface performance, correctness and UX trade-offs to guide a production design.
